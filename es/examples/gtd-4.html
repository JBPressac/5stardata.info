Para ver los datos incrustados en formato RDFa, coloca el cursor sobre las áreas con línea roja punteada. Puedes también ver los datos externamente con Graphite Browser u otras herramientas.

En este ejemplo de 4 estrellas, usamos el vocabulario Meteo para representar el pronóstico de temperatura. Cada pronóstico tiene una URI (por ejemplo, 'pronóstico para Lunes, 17 de Noviembre de 2014'), la cual puede ser utilizada en otro contexto.

Si ves el código fuente, los datos parecen tener una estructura jerárquica, pero esto es una ilusión. Realmente esto solo expresa un conjunto de hechos individuales acerca de las cosas; "#Galway" es "un Lugar", "#Galway" tiene un pronóstico "#pronostico20141117", "#pronostico20141117" es una predicción para "2014-11-17T00:00:00Z" y así sucesivamente. Puedes cambiar el orden de estos hechos sin alterar su significado.
