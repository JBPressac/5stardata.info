Pour voir les données embarquées encodées en RDFa, survolez les zones entourées de pointillés rouges. Vous pouvez aussi voir les données externes dans le navigateur Graphite, ou un autre outil.

Dans cet exemple 4 étoiles, nous avons utilisé le vocabulaire Meteo pour représenter les prévisions des températures. Chaque prévision a un URI (par exemple, 'prévision du lundi 15 novembre 2010'), qui peut être utilisé dans un autre contexte.

Si vous regardez la source, les données ressemblent à une structure hiérarchique, mais cela n’est que pure illusion. En réalité, il s’agit de l’expression d’un ensemble de faits individuels à propos de choses&nbps;; "#Galway" est "un lieu (Place)", "#Galway" a une prévision "#forecast20101113", "#forecast20101113" est une prédiction pour "2010-11-13T00:00:00Z" et ainsi de suite. Vous pouvez changer l’ordre de ces faits, sans pour autant en altérer leur signification.
